## 고객생애가치(CLV)
고객이 평생에 걸쳐 구매할 것으로 예상되는 이익 흐름에 대한 현재가치

### CLV 분석 방안
M : 평균매출액(1인당 연평균 매출액)<br>
r : 고객유지확률(다음 해에도 고객으로 유지될 확률)<br>

* M, r 만 알때 : `CLV = M + Mr + Mr^2 + Mr^3 + ...` = `M(1/1 - r)`

C : 1인당 연평균 유지비용<br>
i : 물가상승률<br>
AC : 최초 고객획득비용<br>

* 다 알때 : `CLV = (M - C)/(1 - r + i) - AC`

#### CLV 기본모형정리


```python
def CLV_r(M, C, r, AC, i):
    result = (M-C)/(1-r+i) - AC
    return result
```
